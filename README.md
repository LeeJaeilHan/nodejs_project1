# nodejs_project1
web application CRUD using file

## features
1. create
2. read
3. update
4. delete
5. refactoring
6. security

## modules
1. html
2. fs
3. url
4. querystring
5. path
6. sanitize-html

## install sanitize-html(in terminal)
* using yarn(package manager for js)
```
yarn init
yarn add sanitize-html
```

## start server
```
node main.js
```

## start server using pm2
```
pm2 start main.js --watch
```
